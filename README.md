Cómo ejecutar el programa:



Guarda tpi1.py y un archivo paises.csv en la misma carpeta.

Abre la terminal (CMD).

Ve a la carpeta donde están los archivos.

Ejecuta:

Python tpi1.py

o

py tpi1.py

El programa mostrará un menú. Escribe el número de la opción que quieras usar y presiona Enter.

⚠️ Si aparece "Archivo no encontrado", verifica que paises.csv esté en la misma carpeta que tpi1.py.
